# 🧙 룬 마법사 엘드린 — 잃어버린 기억

> 웹캠으로 **고개를 움직여** 조종하는 로폴리(low-poly) 3D 오픈월드 어드벤처 게임

기억을 잃은 마법사 **엘드린**이 되어 몽환의 들판을 탐험하고, 떠도는 몬스터들과 싸우며 잃어버린 기억의 조각을 하나씩 되찾는 게임입니다. 키보드 없이 **얼굴 움직임만으로** 캐릭터를 움직이고 마법을 시전할 수 있습니다.

브라우저만 있으면 바로 플레이할 수 있습니다. (설치 불필요)

---

## ✨ 주요 특징

- 🌍 **3D 오픈월드** — 차분하고 몽환적인 분위기의 넓은 들판을 자유롭게 탐험
- 👁 **얼굴 인식 조작** — 웹캠으로 코의 위치를 추적해 고개를 기울이면 그 방향으로 이동 (키보드도 가능)
- ⚔️ **턴제 전투** — 몬스터를 만나면 포켓몬 스타일 1:1 전투로 전환
- 👹 **몬스터 4종** — 슬라임 · 좀비 · 블루 데몬 · 자이언트 (레벨이 다르고, 걷고 공격하는 애니메이션 포함)
- 🗺 **미니맵** — 화면 우측 상단에서 몬스터 위치를 실시간 확인
- 📖 **스토리** — 몬스터를 물리칠 때마다 잃어버린 기억의 조각이 돌아옵니다

---

## 🎮 플레이 방법

### 1. 이동 (월드 탐험)
| 조작 | 동작 |
|---|---|
| **고개 좌 / 우로 기울이기** | 좌 / 우로 이동 |
| **고개 위 / 아래로 움직이기** | 앞 / 뒤로 이동 |
| **W A S D** (키보드) | 동일하게 이동 (얼굴 인식이 어려울 때) |
| **C 키** | 현재 자세를 "정면"으로 재설정 |

> 💡 우측 하단 웹캠 미리보기에 파란 점(코 위치)과 사각형(중심 범위)이 보입니다. 자세가 틀어지면 **C 키**로 다시 맞추세요.

### 2. 전투
몬스터에게 가까이 다가가면 자동으로 전투가 시작됩니다.

1. 화면 중앙에 **방향 룬(▲▼◀▶)** 이 하나씩 떠오릅니다.
2. 그 방향으로 **고개를 움직이면**(또는 방향키 ↑↓←→) 마법이 충전됩니다.
3. 5개를 모두 채우면 **마법이 발사**되어 몬스터를 공격합니다.
4. 몬스터의 반격을 버티며 체력(HP)이 0이 될 때까지 반복하세요.

### 3. 목표
맵에 흩어진 **8마리의 몬스터를 모두 물리치면** 잃어버린 기억이 완성되고 엔딩을 볼 수 있습니다. 곳곳의 **빛나는 표지석**에 다가가면 숨겨진 이야기도 읽을 수 있습니다.

## 🚀 직접 해보기

### 온라인 (가장 쉬움)
Vercel에 배포된 주소로 접속한 뒤, **카메라 권한을 허용**하면 바로 플레이할 수 있습니다.
> ⚠️ 얼굴 인식과 카메라는 보안상 `https` 또는 `localhost`에서만 동작합니다.

### 내 컴퓨터에서 실행
이 게임은 별도 빌드가 필요 없는 정적 웹페이지입니다. 다만 카메라/3D 모델을 쓰기 때문에 **로컬 서버**로 열어야 합니다. (파일을 더블클릭해서 여는 `file://` 방식은 동작하지 않습니다.)

```bash
# 1. 저장소 내려받기
git clone https://github.com/shjuju0721/3D_Game.git
cd 3D_Game

# 2. 간단한 로컬 서버 실행 (둘 중 하나)
python -m http.server 8000        # 파이썬이 있는 경우
# 또는
npx serve                          # Node.js가 있는 경우

# 3. 브라우저에서 열기
#   http://localhost:8000
```

접속 후 **카메라 권한 허용** → "눈을 뜬다" 버튼을 누르면 모험이 시작됩니다.

## 🛠 사용 기술

| 분류 | 사용 기술 |
|---|---|
| 3D 그래픽 | [Three.js](https://threejs.org/) (WebGL) |
| 얼굴 인식 | [MediaPipe Face Landmarker](https://developers.google.com/mediapipe) |
| 후처리 효과 | Bloom(빛 번짐), ACES 톤매핑, 그림자(PCFSoftShadow) |
| 언어 | 순수 HTML / CSS / JavaScript (프레임워크 없음) |
| 배포 | Vercel (정적 호스팅) |

3D 캐릭터·몬스터는 `.glb` 모델 파일을 불러와 애니메이션을 재생합니다.

---

## 📁 파일 구성

```
3D_Game/
├── index.html              # 메인 게임 (3D 오픈월드) ← 여기가 시작 화면
├── battle.html             # 초기 버전 (2D 포켓몬식 전투 게임)
├── Character Animated.glb  # 주인공 모델
├── Slime Enemy.glb         # 몬스터 - 슬라임
├── Zombie.glb              # 몬스터 - 좀비
├── Blue Demon.glb          # 몬스터 - 블루 데몬
└── Giant.glb               # 몬스터 - 자이언트
```

모든 게임 로직(씬 구성, 조작, 전투, 스토리)은 `index.html` 한 파일 안에 들어 있으며, 각 부분에 **한국어 주석**이 달려 있어 코드를 읽으며 학습하기 좋습니다.

## 💡 팁 & 문제 해결

- **캐릭터가 안 움직여요** → 카메라 권한을 허용했는지 확인하고, 얼굴이 웹캠 화면 안에 들어오게 한 뒤 **C 키**로 중심을 다시 맞춰 보세요. 그래도 안 되면 **WASD 키**로 조작할 수 있습니다.
- **처음 로딩이 느려요** → 3D 모델 파일(약 2MB)을 받는 중입니다. 잠시만 기다려 주세요.
- **얼굴 인식이 너무 민감하거나 둔해요** → 조명이 밝은 곳에서, 얼굴이 화면 중앙에 오도록 거리를 조절해 보세요.
