# 🧙 룬 마법사 엘드린 — 잃어버린 기억

> 웹캠으로 **얼굴 움직임과 입·볼·혀 동작**을 인식해 즐기는 로폴리(low-poly) 3D 어드벤처 + 구강 안면 재활(rehabilitation) 게임

기억을 잃은 마법사 **엘드린**이 되어 6개의 구역을 차례로 지나며, **구역마다 다른 얼굴 동작**으로 어둠의 몬스터를 처치하고 잃어버린 기억의 조각을 하나씩 되찾는 게임입니다. 키보드 없이 얼굴만으로 플레이할 수 있습니다.

브라우저만 있으면 바로 플레이할 수 있습니다. (설치 불필요)

---

## ✨ 주요 특징

- 🌍 **3D 오픈월드** — 구역마다 분위기(하늘·땅·안개)가 바뀌는 몽환의 들판
- 👁 **얼굴 인식 조작** — 웹캠으로 코 위치를 추적해 고개를 기울이면 그 방향으로 이동 (W A S D 도 가능)
- 👄 **입 동작 전투** — 구역마다 정해진 얼굴 동작을 **5번 반복**하면 마법이 발사되어 몬스터를 공격
- 🛡 **방어 액션** — 몬스터가 공격할 때 **고개를 숙이면(턱 당기기)** 피해를 크게 줄임
- 👹 **구역별 몬스터 + 보스** — 일반 몬스터를 다 잡으면 구역 보스가 등장, 갈수록 강해짐
- 📖 **스토리 + 성장** — 구역을 클리어할 때마다 기억 조각이 돌아오고 레벨업(체력·마법 위력 상승)
- 📊 **재활 기록** — 구역별 동작 정확도·횟수·시간을 기록하고 CSV로 저장

## 🎮 플레이 방법

| 상황 | 조작 |
|---|---|
| **이동(월드 탐험)** | 고개 기울이기 (또는 W A S D) |
| **공격(전투)** | 그 구역의 **입 동작을 5번 반복** → 마법 발사 |
| **방어(전투)** | 몬스터 공격 때 **고개 숙이기**(턱 당기기) · 또는 ↓ 키 |
| 중심 자세 재설정 | **C** 키 |

> 고개로 **이동**, 입으로 **공격**, 고개 숙여 **방어**. 동작을 크고 정확하게 할수록 데미지가 올라갑니다.

### 구역 순서 (동작 / 인식 방식)

| 구역 | 동작 | 인식 |
|---|---|---|
| 1 새벽의 들판 | 입 벌리기 | `jawOpen` |
| 2 고요한 호숫가 | 입 다물기 | `mouthPress` |
| 3 햇살 언덕 | 웃기 | `mouthSmile` |
| 4 보랏빛 숲 | 입술 오므리기 | `mouthPucker` |
| 5 바람의 협곡 | 볼 부풀리기 | 입술 오므림 + 아래 볼 면적 증가 |
| 6 잿빛 폐허 | 혀 내밀기 | 입 안쪽 색(붉은색·백태) 비율 |

> 1구역은 방어를 못해도 쓰러지지 않는 **연습 구역**이고, 2구역부터는 방어에 실패하면 쓰러져 그 구역을 다시 시작합니다.

## 🚀 직접 해보기

카메라/3D 모델을 쓰기 때문에 **로컬 서버**로 열어야 합니다. (파일 더블클릭 `file://` 방식은 동작하지 않습니다.)

```bash
git clone https://github.com/shjuju0721/3D_Game.git
cd 3D_Game
python -m http.server 8000        # 또는: npx serve
# 브라우저에서 http://localhost:8000 접속 → 카메라 권한 허용
```

> ⚠️ 얼굴 인식과 카메라는 보안상 `https` 또는 `localhost`에서만 동작합니다.

## 🛠 사용 기술

| 분류 | 사용 기술 |
|---|---|
| 3D 그래픽 | [Three.js](https://threejs.org/) (WebGL, Bloom·ACES 톤매핑·그림자) |
| 얼굴/표정 인식 | [MediaPipe Face Landmarker](https://developers.google.com/mediapipe) (랜드마크 + 블렌드셰이프) |
| 혀·볼 인식 | 입 영역 색/면적 분석 (canvas 픽셀 처리) |
| 언어 | 순수 HTML / CSS / JavaScript (프레임워크 없음) |

3D 캐릭터·몬스터는 `.glb` 모델 파일을 불러와 애니메이션을 재생합니다.

## 📁 파일 구성

```
3D_Game/
├── index.html              # 메인 게임 (3D 어드벤처 — 입 동작 전투) ← 시작 화면
├── battle.html             # 초기 버전 (2D 포켓몬식 전투 게임)
├── Character Animated.glb  # 주인공
├── Slime Enemy.glb         # 몬스터 - 슬라임
├── Zombie.glb              # 몬스터 - 좀비
├── Blue Demon.glb          # 몬스터 - 블루 데몬
└── Giant.glb               # 몬스터 - 자이언트
```

모든 게임 로직(씬 구성, 이동, 입 동작 전투, 방어, 스토리)은 `index.html` 한 파일에 들어 있으며 한국어 주석이 달려 있습니다.
